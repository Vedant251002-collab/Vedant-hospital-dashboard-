🏥 Hospital Analytics Dashboard | Power BI

📌 Project Overview
This project is an **interactive Power BI dashboard** designed to analyze hospital operations.  
It provides insights into **patient visits, revenue streams, treatment trends, and performance metrics**, helping healthcare administrators make informed decisions.  

 🎯 Key Features
- 📊 **Patient Visit Analysis** – Track daily, monthly, and yearly visits  
- 💰 **Revenue Dashboard** – Breakdown by payment mode (Cash, Insurance, Online)  
- 🏥 **Department Insights** – , treatment categories  
- 📈 **KPI Cards** – Average daily visits, bed occupancy, revenue growth  
- 🔄 Automated refresh to reduce manual reporting  

 🛠️ Tools & Techniques
- **Power BI Desktop**  
- **Power Query** for data cleaning & ETL  
- **DAX Measures** for custom KPIs (YoY Growth, Avg Daily Visits, Occupancy Rate)  
- **Data Modeling** with relationships between patient, treatment, and finance tables  
- **Interactive Visuals**: KPIs, trend charts, slicers, heat maps
